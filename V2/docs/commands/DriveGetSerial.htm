DriveGetSerial

返回指定驱动器的卷序列号.

Serial := DriveGetSerial(Drive)

参数

Drive

类型: 字符串

The drive letter followed by a colon and an optional backslash, or a UNC name such \server1\share1.

返回值

类型: 整数

This function returns the volume serial number of the specified drive.

ErrorLevel

如果遇到问题, ErrorLevel 设置为 1, 否则设置为 0.
