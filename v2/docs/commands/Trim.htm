Trim

移除字符串开头和/或末尾的某些字符.

Result :=  Trim(String , OmitChars := " `t")
Result := LTrim(String , OmitChars := " `t")
Result := RTrim(String , OmitChars := " `t")

参数

String

类型: 字符串

任何字符串值或变量. 不支持数字.

OmitChars

类型: 字符串

可选的字符列表(区分大小写), 在 String 的开头和/或末尾处移除这些字符. 如果省略, 则移除空格和 tab.

返回值

类型: 字符串

这些函数返回指定字符串修剪后的版本.

示例

text := "  text  "
MsgBox
(
    "No trim:`t'" text "'
    Trim:`t'" Trim(text) "'
    LTrim:`t'" LTrim(text) "'
    RTrim:`t'" RTrim(text) "'"
)
MsgBox LTrim("00000123", "0")
