StrGet

从内存地址复制字符串, 可选地从给定的代码页进行转换.

String := StrGet(Address , Length , Encoding := None)

参数

Address

类型: 整数

要复制的字符串的地址.

Length

类型: 整数

需读取的最大字符数.

默认情况下, 只复制到第一个二进制零. 如果 Length 为负数, 则它的绝对值指示要转换的确切字符数, 包括字符串可能包含的任何二进制零 - 换句话说, 结果始终是具有该长度的字符串.

Encoding

类型: 字符串

源编码; 例如, "UTF-8", "UTF-16" 或 "CP936". 如果 Address 和 Length 都没有指定, 那么数值标识符必须加上 "CP" 前缀. 指定空字符串或 "CP0" 则使用系统默认 ANSI 代码页.

返回值

类型: 字符串

函数返回复制或转换的字符串. 如果正确地指定了源编码, 则返回值总是使用原生编码. 它总是以空终止符终止, 但是空终止符不包括在返回值的 length 中, 除非 Length 为负数, 如前所述.

错误处理

如果检测到无效参数或转换无法执行, 则抛出异常.

备注

注意, 返回值总是使用当前可执行文件的原生编码, 而 Encoding 指定如何解释从给定 地址 读取的字符串. 如果没有指定 Encoding, 只复制字符串而不进行任何转换.

如果需要在代码页之间进行转换, 返回值的长度可能与源字符串的长度不同.

Length 或 Encoding 都可以直接在 Address 后面指定, 但在这种情况下 Encoding 必须是非数字的:

str := StrGet(address, "cp0")  ; 代码页 0, 未指定长度
str := StrGet(address, n, 0)   ; 最大 n 字符, 代码页 0
str := StrGet(address, 0)      ; 最大 0 字符(始终为空)
