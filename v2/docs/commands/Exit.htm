Exit

退出当前线程.

Exit ExitCode

参数

ExitCode

类型: 整数

一个介于 -2147483648 和 2147483647 之间的整数, 在脚本退出时返回给它的调用者. 此代码可以被任何调用脚本的程序访问, 例如另一个脚本(使用 RunWait) 或批处理(.bat) 文件. 如果省略, 则 ExitCode 默认为零. 零一般用来表示成功.

备注

如果脚本不是持续运行的并且这是最后一个线程, 则整个脚本退出.

在其他情况下, Exit 函数会终止当前线程. 换句话说, 由菜单, 计时器或热键子例程直接或间接调用的子程序堆栈全部都会像遇到 Return 那样立即被返回. 如果直接用在这样的子程序内部 -- 而不是由它间接调用的某个子程序中 -- 那么此时 Exit 效果等同于 Return.

使用 ExitApp 来完全终止持续运行的脚本.

相关

ExitApp, OnExit, 函数, Return, Threads, #Persistent

示例

#1

#z::
{
    call_exit
    MsgBox "This MsgBox will never happen because of the Exit."
    call_exit() 
    {
        Exit ; 终止当前函数以及调用它的函数.
    }
}
