ControlGetLineCount

Returns the number of lines in an Edit control.

LineCount := ControlGetLineCount(Control , WinTitle, WinText, ExcludeTitle, ExcludeText)

参数

Control

类型: 字符串, Integer or Object

The control's ClassNN, text or HWND, or an object with a Hwnd property. For details, 请参阅 The Control Parameter.

WinTitle

类型: 字符串, 整数或对象

识别目标窗口的窗口标题或其他条件. 请参阅 WinTitle

WinText

类型: 字符串

如果存在, 此参数必须是目标窗口的单个文本元素的子字符串(和内置的 Window Spy 工具显示的一样). 如果 DetectHiddenText 为 ON, 那么会检测隐藏文本元素.

ExcludeTitle

类型: 字符串

标题中含有此参数值的窗口将被排除.

ExcludeText

类型: 字符串

文本中含有此参数值的窗口将被排除.

返回值

类型: 整数

This function returns the number of lines in an Edit control. All Edit controls have at least 1 line, even if the control is empty.

错误处理

如果遇到问题, ErrorLevel 设置为 1, 否则设置为 0.

An exception is thrown if invalid parameters are detected.

备注

窗口标题和文本是区分大小写的. 除非 DetectHiddenWindows 被打开, 否则不会检测隐藏窗口.
