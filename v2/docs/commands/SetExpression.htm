Var := expression

计算表达式并把结果保存到变量.

Var := expression

参数

Var

保存 表达式 结果的变量的名称.

Expression

有关详情, 请参阅表达式和下面的示例.

注意, 此参数不是可选的. 若要清空该变量, 请指定一个空字符串. 例如: x := ""

备注

单词 true 和 false 是值分别为 1 和 0 的内置变量. 可以使用它们使脚本更易于阅读, 如以下示例所示:

CaseSensitive := false
ContinueSearch := true

可以使用这个函数和任何其他接受 OutputVar 的函数来创建伪数组. 这可以通过让 OutputVar 包含另一个变量的引用来实现, 例如 Array%i% := Var/100 + 5. 有关更多信息, 请参阅数组.

相关

表达式, IF (表达式), 函数, EnvSet, Arrays

示例

赋值一个字符串到变量.

Var := "literal string"

赋值一个数字到变量.

Var := 3

计算净价并将结果存储在 Var.

Var := Price * (1 - Discount/100)

判断一个表达式的真假, 并将结果(1 为真或 0 为假) 存储在 Finished.

Finished := not Done or A_Index > 100
if not Finished
{
    FileAppend NewText "`n", TargetFile
    return
}
else
    ExitApp
