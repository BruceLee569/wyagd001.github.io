GuiCtrlFromHwnd

检索与指定的 HWND 关联的 GUI 控件的 GuiControl 对象.

GuiControlObj := GuiCtrlFromHwnd(Hwnd)

参数

Hwnd

类型: 整数

GUI 控件的窗口句柄(HWND). 可以使用 Gui.Add 创建这样的控件.

Return Value

类型: 对象

函数返回对现有 GuiControl 对象的引用. 该对象提供了用于修改 GUI 控件和检索有关控件的信息的方法和属性.

备注

例如, 能通过 GuiCtrl.Hwnd, MouseGetPos 或 OnMessage 来检索 GUI 控件的 HWND.

相关
