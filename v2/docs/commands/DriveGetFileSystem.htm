DriveGetFileSystem

返回指定驱动器的文件系统的类型.

FileSystem := DriveGetFileSystem(Drive)

参数

Drive

类型: 字符串

驱动器字母后跟着冒号和可选的反斜杠, 或 UNC 名称, 如 \server1\share1.

返回值

类型: 字符串

函数返回 Drive 的文件系统的类型: FAT, FAT32, NTFS, CDFS(通常表示 CD) 或 UDF(通常表示 DVD). 如果 Drive 中不包含格式化的媒体, 返回值置空且 ErrorLevel 设置为 1.

ErrorLevel

如果遇到问题, ErrorLevel 设置为 1, 否则设置为 0.
