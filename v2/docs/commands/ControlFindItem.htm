ControlFindItem

Returns the entry number of a ListBox or ComboBox that is a complete match for the specified string.

FoundItem := ControlFindItem(String, Control , WinTitle, WinText, ExcludeTitle, ExcludeText)

参数

String

类型: 字符串

The string to find. The search is case-insensitive. Unlike ControlChooseString, the entry's entire text must match, not just the leading part.

Control

类型: 字符串, Integer or Object

The control's ClassNN, text or HWND, or an object with a Hwnd property. For details, 请参阅 The Control Parameter.

WinTitle

类型: 字符串

A window title or other criteria identifying the target window. See WinTitle.

WinText

类型: 字符串

If present, this parameter must be a substring from a single text element of the target window (as revealed by the included Window Spy utility). Hidden text elements are detected if DetectHiddenText is ON.

ExcludeTitle

类型: 字符串

Windows whose titles include this value will not be considered.

ExcludeText

类型: 字符串

Windows whose text include this value will not be considered.

返回值

类型: 整数

This function returns the entry number of a ListBox or ComboBox that is a complete match for String. The first entry in the control is 1, the second 2, and so on. If no match is found, the return value is blank and ErrorLevel is set to 1.

Error Handling

如果遇到问题, ErrorLevel 设置为 1, 否则设置为 0.

An exception is thrown if invalid parameters are detected.

备注

To improve reliability, a delay is done automatically after each use of this function. That delay can be changed via SetControlDelay or by assigning a value to A_ControlDelay. For details, 请参阅 SetControlDelay remarks.

Window titles and text are case sensitive. Hidden windows are not detected unless DetectHiddenWindows has been turned on.
