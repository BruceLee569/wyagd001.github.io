IsSet

如果指定变量已经被赋值, 则返回一个非零数字.

VarIsSet := IsSet(&Var)

参数

&Var

类型: VarRef

变量的引用. 例如: IsSet(&MyVar).

注意: 在直接检查变量时, 需要使用引用操作符(&), 但在传递 包含 VarRef 的变量时不需要使用.

返回值

类型: 整数(布尔值)

如果 Var 已经被赋值, 函数返回 1(true), 否则返回 0(false).

备注

未赋值的变量也称为未初始化变量. 默认情况下, 试图读取一个未初始化的变量会导致异常抛出. IsSet 可以用来避免这种情况, 例如在第一次使用全局变量或静态变量时是否需要初始化.

在函数内部检查全局变量时, 一般需要声明该变量. 否则, 使用引用操作符(&) 会创建一个局部变量. 如果要在函数内部给全局变量赋值, 这种声明是必须的.

注意: 静态初始化 如 static my_static_array := [] 只被计算一次, 即在执行过程中第一次到达时, 因此通常不需要使用 IsSet.

已知限制: 如果 MsgBox 警告模式生效, 触发给定变量的警告消息的副作用是初始化该变量以防止进一步的警告. 这将导致 IsSet 为后续调用返回 true.

相关

ByRef 参数

示例

如果 MyVar 已经被赋值, 则报告一个非零的数字.

Loop 2
    if !IsSet(&MyVar)  ; 是否是第一次 "使用" MyVar?
        MyVar := A_Index  ; 第一次 "使用" 时进行初始化.
MsgBox Function1(&MyVar)
MsgBox Function2(&MyVar)

Function1(&Param)  ; ByRef 参数.
{
    if IsSet(&Param)  ; 传递引用给目标 Param.
        return Param  ; ByRef 参数会自动解引用.
    else
        return "unset"
}
Function2(Param)
{
    if IsSet(Param)  ; 传递包含 VarRef 的 Param.
        return %Param%  ; 显式解引用 Param.
    else
        return "unset"
}
