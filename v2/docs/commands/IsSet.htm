IsSet

如果指定变量已经被赋值, 则返回一个非零数字.

IsSet := IsSet(Var)

参数

Var

类型: 变量

变量的引用. 例如: IsSet(MyVar).

返回值

类型: 整数(布尔值)

如果 Var 已经被赋值, 函数返回 1(true), 否则返回 0(false).

备注

未赋值的变量也称为未初始化变量. 默认情况下, 试图访问一个未初始化的变量(除了这个函数之外的任何方式) 会导致显示警告, 但是未来的版本会以错误代替这个警告

IsSet 可用于确定一个变量是否需要初始化, 例如在首次使用时初始化一个全局或静态变量.

注意: 静态初始化 如 static my_static_array := [] 只被\计算一次, 即在执行过程中第一次到达时, 因此通常不需要使用 IsSet.

已知限制: 如果 MsgBox 警告模式生效, 触发给定变量的警告消息的副作用是初始化该变量以防止进一步的警告. 这将导致 IsSet 为后续调用返回 true.

相关

ByRef 参数

示例

#1

MsgBox Function(MyVar)

 Function(ByRef Param)
{
    return IsSet(Param)
