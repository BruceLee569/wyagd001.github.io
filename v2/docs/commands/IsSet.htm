IsSet

Returns a non-zero number if the specified variable has been assigned a value.

IsSet := IsSet(Var)

Parameters

Var

Type: Variable

A reference to the variable. For example: IsSet(MyVar).

Return Value

Type: Integer (boolean)

This function returns 1 (true) if Var has been assigned a value, otherwise 0 (false).

Remarks

A variable which has not been assigned a value is also known as an uninitialised variable.

Uninitialised variables contain an empty string by default, and some script authors may choose to rely on this. However, if #Warn UseUnset has been used, accessing an uninitialised variable by almost any other means other than IsSet may trigger a warning message. IsSet can be used to determine whether a variable requires initialization, such as for initializing a global or static variable on first use.

Known limitation: If the MsgBox warning mode is in effect, triggering the warning message for a given variable has the side-effect of initializing that variable to prevent further warnings. This causes IsSet to return true for subsequent calls.
