ControlGetPos

获取控件的位置和大小.

ControlGetPos X, Y, Width, Height, Control, WinTitle, WinText, ExcludeTitle, ExcludeText

参数

X, Y

类型: 变量

用于存储 Control 左上角的 X 和 Y 坐标(英寸) 的变量的名称. 这些坐标相对于目标窗口的客户端区域, 因此与 ControlMove 使用的坐标相同.

如果省略 X 或 Y, 则不会存储相应的值.

Width, Height

类型: 变量

用于存储 Control 的宽度和高度(单位为英寸) 的变量的名称. 如果省略, 则不会存储相应的值.

Control

类型: 字符串, 整数或对象

控件的 ClassNN, 文本或 HWND, 或具有 Hwnd 属性的对象. 有关详情, 请参阅 Control 参数.

WinTitle

类型: 字符串, 整数或对象

识别目标窗口的窗口标题或其他条件. 请参阅 WinTitle.

WinText

类型: 字符串

如果存在, 此参数必须是目标窗口的单个文本元素的子字符串(和内置的 Window Spy 工具显示的一样). 如果 DetectHiddenText 为 ON, 那么会检测隐藏文本元素.

ExcludeTitle

类型: 字符串

标题中含有此参数值的窗口将被排除.

ExcludeText

类型: 字符串

文本中含有此参数值的窗口将被排除.

备注

如果没有找到匹配的窗口或控件, 则将输出变量置空.

与更改控件的函数不同, ControlGetPos 没有自动延迟(不受 SetControlDelay 影响).

窗口标题和文本是区分大小写的. 除非 DetectHiddenWindows 被打开, 否则不会检测隐藏窗口.

相关

ControlMove, WinGetPos, Control 函数

示例

#1: 这个示例会持续地更新和显示当前鼠标光标下的控件的名称和位置:

Loop
{
    Sleep 100
    MouseGetPos ,, WhichWindow, WhichControl
    ControlGetPos x, y, w, h, WhichControl, "ahk_id " WhichWindow
    ToolTip WhichControl "`nX" X "`tY" Y "`nW" W "`t" H
}
