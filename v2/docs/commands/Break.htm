Break

退出(终止) loop. 在各种 loop 中都是有效的.

Break LoopLabel

如果指定了, LoopLabel 标识该语句应该应用于哪个 loop; 可为标签名称或数字嵌套层级. 如果省略或为 1, 则此语句适用于包含它的最内层循环. 如果指定了一个标签, 标签必须直接指向循环语句.

LoopLabel 必须为常量 - 不支持变量和表达式, 是一个用圆括号括起来的原义数字或引号括起来的字符串. 例如: break("outer")

鼓励使用 Break 和 Continue 代替 goto, 因为它们通常让脚本更容易阅读和维护.

相关

Continue, Loop, While-loop, For-loop, Blocks, Labels

示例

Loop
{
    ...
    if var > 25
        break
    ...
    if var <= 5
        continue
}
; 从套嵌循环中退出 outer(外部) 循环.
outer:
Loop 3
{
    x := A_Index
    Loop 3
    {
        if (x*A_Index = 6)
            break outer  ; 等同于 break 2 或 goto break_outer.
        MsgBox x "," A_Index
    }
}
break_outer: ; 用于 goto.
