IsByRef

如果函数的 ByRef 参数由指定的变量提供, 则返回一个非零数字.

IsByRef := IsByRef(ParameterVar)

参数

ParameterVar

类型: 变量

变量的引用. 例如: IsByRef(MyParameter).

返回值

类型: 整数(布尔值)

如果 ParameterVar 是 ByRef 参数, 且调用者传递了这个参数, 则函数返回 1(true); 否则当 ParameterVar 为其他类型参数时返回 0(false).

相关

ByRef 参数

示例

MsgBox Function(MyVar)

Function(ByRef Param)
{
    return IsByRef(Param)
}
