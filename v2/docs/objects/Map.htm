Map 对象

Map(映射) 对象将一组称为 键 的值关联或 映射 到另一组值. 键及其映射到的值称为键-值对. 映射可以包含任意数量的键值对, 但是每个键必须是惟一的.

键可以是任何整数, 对象 引用以 null 结尾的字符串. 字符串键的比较是区分大小写的, 而对象是通过引用/地址进行比较的. 浮点数键被自动转换成字符串.

映射的最简单用法是通过隐式的 __Item 属性检索或设置键-值对, 只需在映射对象后面的方括号中写入键. 例如:

clrs := Map()
clrs["Red"] := "ff0000"
clrs["Green"] := "00ff00"
clrs["Blue"] := "0000ff"
for clr in Array("Blue", "Green")
    MsgBox clrs[clr]

下面的文档使用 aMap 作为任何映射对象的占位符. 注意, "map" 不应该用作变量名, 因为它是用来引用 Map 类本身.

Map 对象基于 Map.Prototype, 其又基于 Object.Prototype. 除了继承自 Object 的方法和属性外, 映射对象还具有以下预定义的方法和属性:

方法:

  • Clear
  • Clone
  • Delete
  • Has
  • Set
  • __Enum

属性:

  • Count
  • Capacity
  • CaseSense
  • __Item

函数:

  • Map

方法

Clear

从映射中删除所有键-值对.

aMap.Clear()

Clone

返回对象的一个浅拷贝.

Clone := aMap.Clone()

所有键-值对都被复制到新映射. 对象 引用 被复制(就像普通的赋值一样), 而不是对象本身.

根据 Obj.Clone 来复制自有的属性, 方法和基.

Delete

从映射中删除键-值对.

aMap.Delete(Key)
Key

类型: 整数, 对象或字符串

任意一个键. 如果映射不包含此键, 则抛出异常.

返回值

删除的值.

Has

如果 Key 在映射中有关联的值, 则返回 true, 否则返回 false.

aMap.Has(Key)

Set

设置零个或多个项目.

aMap.Set(Key, Value, Key2, Value2, ...)

这相当于用 aMap[Key] := Value 来设置每个项目, 除了 __Item 没有被调用和 Capacity 会被自动调整, 以避免在一次调用中扩展多次.

返回 map 对象.

__Enum

枚举键-值对.

For Key , Value in aMap

返回一个新的枚举器. 该方法通常不直接调用. 而是将映射对象被直接传递给 for-loop, 其调用一次 __Enum 然后循环的每次迭代都调用一次枚举器. 每次对枚举器的调用都会返回下一个键和/或值. for-loop 的变量对应于枚举器的参数, 它们是:

Key

类型: 整数, 对象或字符串

键.

Value

值.

属性

Count

检索映射中存在的键-值对的数量.

Count := aMap.Count

Capacity

检索或设置映射的当前容量.

aMap.Capacity := MaxItems
MaxItems := aMap.Capacity
MaxItems

类型: 整数

在必须自动扩展前, 映射可以包含的最大键-值对数量. 如果设置的值小于当前键-值对数量, 则使用设置的值并释放未使用的空间.

CaseSense

检索或设置映射的大小写敏感性设置.

aMap.CaseSense := Setting
Setting := aMap.CaseSense
Setting

类型: 字符串

One of the following values:

"On": Key lookups are case sensitive. This is the default setting.

"Off": The letters A-Z are considered identical to their lowercase counterparts.

"Locale": Key lookups are case insensitive according to the rules of the current user's locale. For example, most English and Western European locales treat not only the letters A-Z as identical to their lowercase counterparts, but also non-ASCII letters like Ä and Ü as identical to theirs. Locale is 1 to 8 times slower than Off depending on the nature of the strings being compared.

在赋值时, 也可以使用字符串 "1" 和 "0".

无论 A_StringCaseSense 的值是多少, 默认值都是 "On".

如果 Map 不是空的, 尝试为该属性赋值将导致抛出异常.

__Item

检索或设置键-值对的值.

Value := aMap[Key]
aMap[Key] := Value
Key

类型: 整数, 对象或字符串

检索值时, Key 必须是先前与另一个值相关联的唯一值. 如果 Key 在映射中没有关联的值, 则抛出异常.

当赋值时, Key 可以是与 Value 相关联的任何值; 换句话说, key 用于以后访问 Value. 浮点数键被自动转换成字符串.

如上所示, 属性名 __Item 通常被省略, 但在覆盖属性时使用.

函数

Map

创建一个 Map 并设置项目.

aMap := Map(Key1, Value1, Key2, Value2, ...)

这相当于用 aMap := Map.new() 创建一个 Map, 设置它的初始 Capacity, 然后用 aMap[Key] := Value 设置每个项目. 避免了多次扩展, 但不能保证 Capacity 的精确性.
