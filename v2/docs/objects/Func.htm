Func 对象

表示可以被脚本调用的自定义或内置的函数. Func 返回这种类型的对象(或 Closure(闭包) 对象具有与其相同的属性).

有关其他可以像函数一样调用的对象的信息, 请参阅函数对象.

对 Func 对象的引用也被称为 函数引用. 要获取函数引用, 请使用 Func 函数, 如下面的例子所示:

; 检索对名为 "StrLen" 的函数引用.
fn := Func("StrLen")

; 显示函数的信息.
MsgBox fn.Name "() is " (fn.IsBuiltIn ? "built-in." : "user-defined.")

对象成员

属性:

  • Name: 返回函数的名称.
  • IsBuiltIn: 内置函数返回 true, 否则返回 false.
  • IsVariadic: 当函数为可变参数时返回 true, 否则返回 false.
  • MinParams: 返回所需参数的数量.
  • MaxParams: 对于用户定义函数返回正式声明的参数数目, 对于内置函数返回最大的参数数目.

方法:

  • Call: 调用函数.
  • Bind: 绑定参数到函数并返回绑定函数对象.
  • IsByRef: 确定参数是否为 ByRef 类型.
  • IsOptional: 确定参数是否是可选的.

Call

调用函数.

Func.Call(Param1, Param2, ...)
%Func%(Param1, Param2, ...)
Param1, Param2, ...
参数和返回值由函数定义.

Bind

为函数绑定参数并返回绑定函数对象.

BoundFunc := Func.Bind(Param1, Param2, ...)
Param1, Param2, ...
任意数量的参数.

有关详情和示例, 请参阅绑定函数对象.

Name

返回函数的名称.

FunctionName := Func.Name

IsBuiltIn

内置函数返回 true, 否则返回 false.

Boolean := Func.IsBuiltIn

IsVariadic

当函数为可变参数时返回 true, 否则返回 false.

Boolean := Func.IsVariadic

MinParams

返回函数必要参数的数目.

ParamCount := Func.MinParams

MaxParams

对于用户定义函数返回正式声明的参数数目, 对于内置函数返回最大的参数数目.

ParamCount := Func.MaxParams

如果函数是可变参数的, ParamCount 表示函数在不考虑 "variadic*" 可变参数的情况下可接受的最大参数数目.

IsByRef

确定一个参数是否为 ByRef(按地址传递) 类型.

Boolean := Func.IsByRef(ParamIndex)
ParamIndex

类型: 整数

可选: 基于 1 的参数索引. 如果省略, Boolean 表示此函数是否含有 ByRef 参数.

返回指示参数是否为 ByRef 的布尔值. 如果函数为内置函数或 ParamIndex 无效时, 则抛出异常.

IsOptional

确定一个参数是否是可选的.

Boolean := Func.IsOptional(ParamIndex)
ParamIndex

类型: 整数

可选: 基于 1 的参数索引. 如果省略, Boolean 表示此函数是否含有可选参数.

返回指示参数是否为可选参数的布尔值. 如果 ParamIndex 无效, 则抛出异常.

如果此函数是可变参数的, 那么参数可以不进行正式声明. 支持内置函数.
